# Portfolio Dashboards

A collection of interactive data dashboards built by **Ezequiel Bassa** — Senior Data Scientist & Sociologist. Each project combines synthetic or real-world datasets with professional-grade visualizations, deployed as live web applications and embedded in [ezequielbassa.com](https://ezequielbassa.com).

---

## Dashboards

### 1 — Maritime Workplace Climate Survey 2026
> `maritime-climate-dashboard/`

An interactive R Shiny dashboard analyzing workplace climate across a Tier-1 Global Maritime Logistics Company operating in 10 countries throughout the Americas.

| | |
|---|---|
| **Live app** | [ezequielbassa.shinyapps.io/maritime-climate-dashboard](https://ezequielbassa.shinyapps.io/maritime-climate-dashboard) |
| **Dataset** | 12,500 employees × 30 columns (synthetic, seed=42) |
| **Stack** | Python · R · Shiny · bslib · plotly · DT |
| **Tabs** | Overview · Climate Indicators · Department Analysis · Regional View · Workforce Profile · Open Comments · Americas Map |
